# Project Instructions

Before working on this project, read `PROJECT_CONTEXT.md`.

This project is StartFlow, a behavior-start experiment tool. The current demo is meant to test whether a tiny, concrete, 2-minute action chain helps users begin something they have been avoiding.

Keep changes focused on the MVP experiment loop:

1. User enters one thing they want to start but have not started.
2. The app generates a 2-minute chain of 3 tiny actions.
3. User can optionally refine the chain by describing what does not fit their current situation.
4. User executes for 2 minutes.
5. User submits feedback.
6. Data is saved for validation.

Do not turn this version into a general project-management, todo-list, habit-tracking, coaching, or workflow-builder product unless explicitly requested.
